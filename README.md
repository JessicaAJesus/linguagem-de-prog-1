# Informações sobre o repositório
Este repositório reúne exercícios desenvolvidos na disciplina Linguagem de Programação 1. A disciplina introduz os fundamentos da programação orientada a objetos, capacitando os alunos a analisar problemas de baixa complexidade e a projetar e implementar soluções de software sob tal perspectiva, além de fornecer uma base para o aprofundamento na linguagem estudada e em outras linguagens orientadas a objetos. Os exercícios estão organizados por aula, e cada pasta contém um README.md com a descrição das atividades e os respectivos arquivos .java.
