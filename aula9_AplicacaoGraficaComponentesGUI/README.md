# Exercício 1 – Calculadora com Java Swing

## Descrição

Desenvolvimento de uma calculadora gráfica utilizando Java Swing. A aplicação recebe dois valores numéricos e permite realizar operações de soma, subtração, multiplicação, divisão e resto da divisão. O programa realiza validações de entrada antes de efetuar os cálculos e apresenta o resultado por meio de uma caixa de diálogo.

## Conceitos praticados

- Java Swing
- Componentes gráficos (JFrame, JTextField, JComboBox, JButton)
- Tratamento de eventos
- Validação de entrada
- Operadores aritméticos

# Exercício 2 – Conversor de Temperaturas

## Descrição

Aplicação desenvolvida em Java Swing para conversão entre as escalas Celsius, Fahrenheit e Kelvin. A conversão é executada automaticamente conforme a opção selecionada pelo usuário através de RadioButtons.

## Conceitos praticados

- Java Swing
- RadioButton
- Eventos de interface
- Conversões matemáticas
- Manipulação de componentes gráficos

# Exercício 3 – Simulador de Compra com Frete

## Descrição

Aplicação gráfica que calcula o valor final de uma compra considerando descontos para clientes fidelidade, estado de destino e modalidade de envio. O sistema exibe separadamente o valor do produto, do frete e o total da compra.

## Conceitos praticados

- Java Swing
- CheckBox
- ComboBox
- Regras de negócio
- Cálculos condicionais
- Labels dinâmicas

# Exercício 4 – Estrutura de Menus

## Descrição

Desenvolvimento da estrutura de menus de uma aplicação Java Swing contendo menus, submenus e abertura de diferentes janelas, simulando a navegação de um sistema de cadastro.

## Conceitos praticados

- Java Swing
- JMenuBar
- JMenu
- JMenuItem
- Múltiplas janelas
- Organização de interface gráfica
