# Exercício 1 – Pesquisa em Texto

## Descrição

Desenvolvimento de uma aplicação para armazenamento e pesquisa de cadeias de caracteres em um texto, permitindo buscas gerais, no início e no final do texto por meio da sobrescrita de métodos em classes especializadas.

## Conceitos praticados

- Herança
- Sobrescrita de métodos
- Método final
- Polimorfismo
- Manipulação de strings
- Menus e estruturas de repetição

# Exercício 2 – Cálculo de Áreas com Classe Abstrata

## Descrição

Aplicação para cálculo da área de retângulos e círculos, utilizando uma classe abstrata comum para representar formas geométricas e armazenar propriedades compartilhadas, como cor e área.

## Conceitos praticados

- Classes abstratas
- Herança
- Construtores
- Polimorfismo
- Encapsulamento

---

# Exercício 3 – Cálculo de Áreas com Interface

## Descrição

Refatoração da aplicação anterior para substituir o método abstrato de cálculo de área por uma interface, mantendo a funcionalidade de cálculo e exibição das propriedades das formas geométricas.

## Conceitos praticados

- Interfaces
- Polimorfismo
- Separação de responsabilidades
- Refatoração de código
