# Exercício 1 – Menu de Operações

## Descrição

Desenvolvimento de um programa que realiza operações de arredontar número, calcular raiz quadrada e calcular a potência, de acordo com a opção digitada pelo usuário.

## Conceitos praticados

- Estruturas condicionais e de repetição
- Operadores aritiméticos
- Funções matemáticas da API Java (classe Math)

# Exercício 2 – Operações no texto

## Descrição

Aplicação desenvolvida em Java para realizar operações em um texto, apresentando ao usuário as modificações realizadas e resultados de operações como extração de frase, substituição e alteração de ocorrências e quantificação de palavras e caracteres.

## Conceitos praticados

- Manipulação de cadeias de caracteres (classe String)
- Estruturas condicionais

# Exercício 3 – Operações em data
## Descrição

Aplicação que realiza operações de adicionar ou subtrair dias, meses ou anos de uma data fornecida pelo usuário.

## Conceitos praticados

- Manipulação de data e hora
- Encapsulamento (métodos Getters e Setters)
- Formatação de dados
