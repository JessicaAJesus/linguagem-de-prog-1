# Exercício – Controle de Estoque com Testes Unitários

## Descrição

Desenvolvimento de uma aplicação para gerenciamento e análise de estoque de produtos, classificando-os conforme sua situação (insuficiente, excedente, zerado ou adequado) a partir de limites mínimo e máximo definidos. Além da implementação das classes do domínio e das regras de negócio, foram criados testes unitários com JUnit para validar o comportamento esperado da aplicação e o tratamento de casos de borda.

## Conceitos praticados

- Modelagem orientada a objetos
- Associações entre classes
- Coleções genéricas (`List`)
- Encapsulamento
- Regras de negócio
- Testes unitários com JUnit
- Verificação de casos de teste e condições limite
