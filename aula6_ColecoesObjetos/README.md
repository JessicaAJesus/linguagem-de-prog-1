# Exercício 1 – Lista de Produtos

## Descrição

Desenvolvimento de uma aplicação para gerenciamento de produtos utilizando uma lista dinâmica, permitindo inserção, consulta, substituição e remoção de itens por nome ou posição.

## Conceitos praticados

- Collections Framework (`ArrayList`)
- Encapsulamento e organização em classes
- Manipulação de listas
- Busca e atualização de elementos
- Menus e estruturas de repetição

# Exercício 2 – Fila de Pessoas

## Descrição

Aplicação para controle de uma fila de pessoas, respeitando a ordem de chegada e saída dos elementos e permitindo inserção, remoção e visualização da fila.

## Conceitos praticados

- Estrutura de dados fila
- Collections Framework (`LinkedList`)
- Princípio FIFO
- Manipulação de coleções

---

# Exercício 3 – Operações com Conjuntos

## Descrição

Sistema para manipulação de dois conjuntos de produtos, realizando operações de união, interseção, diferença e verificação de subconjuntos.

## Conceitos praticados

- Collections Framework (`TreeSet`)
- Teoria dos conjuntos
- Ordenação automática
- Operações entre coleções

---

# Exercício 4 – Mapa de Produtos e Preços

## Descrição

Aplicação para cadastro e consulta de produtos e seus preços, permitindo buscas, alterações e remoções por chave, além da visualização de nomes, valores e pares chave-valor.

## Conceitos praticados

- Collections Framework (`HashMap`)
- Associação chave-valor
- Busca e atualização de dados
- Manipulação de mapas
