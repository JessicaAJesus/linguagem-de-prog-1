# Exercício – Tratamento de Exceções

## Descrição

Aprimoramento das aplicações desenvolvidas anteriormente por meio da identificação de operações suscetíveis a falhas e da implementação de tratamento de exceções. Foram tratados erros relacionados à conversão e validação de dados de entrada, operações matemáticas e regras de negócio, utilizando as estruturas `try`, `catch` e `finally`, além do lançamento explícito de exceções com `throw` quando necessário.

## Conceitos praticados

- Tratamento de exceções
- Classes de exceção da API Java
- Instruções `try`, `catch` e `finally`
- Lançamento de exceções com `throw`
- Validação de dados e regras de negócio
